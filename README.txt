== Description ==
Hoplit Post Suggest is a straightforward plugin that facilitates the submission of articles ready for publication on your WordPress site. It provides a user-friendly form for users to suggest articles with titles and content.

== Installation ==
1. Download the plugin from the Plugins section in your WordPress dashboard or download it from the WordPress.org repository.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Utilize the `[post_suggest_form]` shortcode to display the article suggestion form on any page or post.

== Usage ==
After activating the plugin, insert the `[post_suggest_form]` shortcode into the content of the page or post where you want to display the article suggestion form. Users can input the title and content of their suggested articles.

Once a user submits a suggestion, they will receive feedback indicating whether the submission was successful or not.

== FAQ ==
= How can users submit articles for consideration? =
Users can submit articles by filling out the form generated with the `[post_suggest_form]` shortcode.

= How do I know if an article suggestion was successful? =
After submission, users will receive feedback on the success or failure of their suggestion.
